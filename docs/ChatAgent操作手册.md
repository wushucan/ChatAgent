# ChatAgent 操作手册

## 启动

```bash
conda activate langgraph_env
python run_chat.py
```

启动后出现桌面窗口，左侧为会话列表面板，右侧为对话区域。

## 功能说明

### 多会话管理

- **新建会话**：点击侧栏顶部的 `+` 按钮
- **切换会话**：点击侧栏中的会话条目
- **删除会话**：鼠标悬停会话条目，点击出现的 `×` 按钮（需确认）
- **自动标题**：每次对话完成后，AI 自动根据对话内容生成会话标题

### 消息操作

- **复制消息**：选中消息文本后复制（鼠标选中文字，Ctrl+C）
- **删除消息**：右键点击消息气泡 → 「删除这条消息」
- **编辑消息**：暂不支持编辑已发送消息

### 联网搜索

- 输入栏左侧的 🌐 开关控制搜索功能
- 开启后 AI 可根据需要自动搜索互联网获取最新信息
- 需要配置 `TAVILY_API_KEY` 环境变量

### 文档知识库（RAG）

- 点击侧栏文档库的 `+` 按钮上传文档
- 支持格式：PDF、TXT、MD、DOCX
- 上传后 AI 可在对话中搜索文档内容回答
- 鼠标悬停文档条目可删除

### 跨会话记忆

- AI 会自动记忆对话中用户提到的个人信息和偏好
- 侧栏底部可查看已保存的记忆列表
- 支持手动删除记忆

## 配置

编辑项目根目录的 `config.json`：

```json
{
    "api_key": "sk-xxxxxxxxxxxxxxxx",
    "model": "deepseek-v4-flash",
    "system_prompt": "你是DeepSeek AI助手..."
}
```

也可通过 `.env` 文件配置（优先级：config.json > .env）：

```
DEEPSEEK_API_KEY=sk-xxx
DEEPSEEK_MODEL=deepseek-v4-flash
TAVILY_API_KEY=tvly-xxx
```

## 数据文件

所有数据保存在 `sessions/` 目录下：

```
sessions/
├── index.json           # 会话索引
├── {session_id}.json    # 每个会话的消息记录
├── memories.json        # 跨会话记忆
└── .rag/                # 文档知识库索引
```

## 常见问题

**Q: 启动后窗口空白**
A: 检查控制台输出是否有报错。确认 pywebview 依赖已安装。

**Q: 发送消息后无响应**
A: 检查 config.json 或 .env 中的 API Key 是否正确。

**Q: 联网搜索不可用**
A: 检查是否设置了 TAVILY_API_KEY，以及输入栏左侧的搜索开关是否开启。

**Q: Embedding 模型加载慢**
A: 首次启动需要下载模型文件（约 30MB），后续启动会在后台加载（约 1-2 秒）。
